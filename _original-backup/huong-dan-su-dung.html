1.Tổng quan

PDF Fusion Smart Pro là công cụ xử lý PDF ngay trên trình duyệt. Bạn có thể gộp nhiều PDF, tách trang, ký tài liệu, thêm watermark chữ, watermark logo, tạo QR, OCR văn bản, quản lý trang và xử lý hàng loạt mà không cần cài phần mềm nặng.

Điểm quan trọng: file được xử lý trực tiếp trong trình duyệt của bạn. Quy trình sử dụng phù hợp cho tài liệu cá nhân, hợp đồng nội bộ, biểu mẫu, scan, hóa đơn, hồ sơ học tập và các tài liệu văn phòng thường ngày.

Giao diện chính được chia thành hai khu vực: bên trái là danh sách file, bên phải là các tab chức năng. Các tab thường dùng nhất là Xuất, Tách PDF, Chữ ký, Pages, OCR, Batch và Tools.

2.Quy trình nhanh cho người mới

  1. Nhấn hoặc kéo thả file vào vùng chọn file.
  2. Sắp xếp thứ tự file bằng cách kéo biểu tượng ☰.
  3. Chọn tab chức năng cần dùng: gộp, tách, ký, watermark hoặc OCR.
  4. Thiết lập tùy chọn như dải trang, màu watermark, logo, chữ ký hoặc QR.
  5. Bấm nút xử lý chính, ví dụ Xuất PDF, Tách & Tải về hoặc Đóng dấu ký & Xuất PDF.
  6. Mở file kết quả và kiểm tra lại từng trang trước khi gửi hoặc lưu trữ chính thức.

3.Thêm file vào dự án

Bạn có thể thêm nhiều loại file:

  • PDF: dùng để gộp, tách, ký, watermark, OCR, quản lý trang.
  • DOCX: app sẽ đọc phần text cơ bản và đưa vào PDF.
  • PNG/JPG: app sẽ chuyển ảnh thành trang PDF.

Sau khi thêm file, mỗi file sẽ có các tùy chọn riêng:

Tùy chọnÝ nghĩa
Dải trangChọn các trang cần dùng, ví dụ 1-3,5,9-10. Để trống nghĩa là dùng tất cả trang.
XoayXoay toàn bộ trang của file đó trước khi xuất.
Bật/TắtFile tắt sẽ không được đưa vào kết quả xuất.
Mẹo: nên đặt tên file rõ ràng như 01-hop-dong.pdf, 02-phu-luc.pdf để thao tác dễ hơn.

4.Gộp PDF

Chức năng gộp PDF nằm ở tab Xuất. Đây là thao tác dùng khi bạn muốn tạo một file PDF duy nhất từ nhiều file đầu vào.

  1. Thêm các file cần gộp.
  2. Kéo thả để sắp xếp đúng thứ tự.
  3. Nếu chỉ cần một số trang của từng file, nhập dải trang vào ô của file đó.
  4. Chọn có đánh số trang hay không.
  5. Chọn có thêm trang trắng sau file có số trang lẻ hay không.
  6. Bật watermark, logo, QR hoặc chữ ký nếu cần.
  7. Bấm Xuất PDF.

Ví dụ: nếu file A nhập 1-2 và file B nhập 3,5, kết quả sẽ gồm trang 1-2 của file A và trang 3,5 của file B theo đúng thứ tự danh sách.

5.Tách PDF

Tab Tách PDF có hai chế độ:

Tách theo dải trang

Nhập các dải trang cách nhau bằng dấu phẩy. Mỗi dải sẽ tạo một file PDF riêng.

  • 1-2,4-5: tạo 2 file, một file trang 1-2 và một file trang 4-5.
  • 5-10: tạo 1 file gồm trang 5 đến 10.
  • 1,3,7-9: tạo 3 file riêng.

Tách từng trang

Chế độ này xuất mỗi trang thành một file PDF riêng. Nếu chỉ muốn tách vài trang, nhập giới hạn như 1-5. Nếu để trống, app sẽ tách toàn bộ trang.

6.Watermark chữ

Watermark chữ dùng để đóng dấu như CONFIDENTIAL, DRAFT, INTERNAL USE ONLY hoặc tên công ty lên tài liệu.

  • Bật watermark chữ: kích hoạt watermark.
  • Nội dung: nhập chữ muốn đóng lên PDF.
  • Cỡ chữ: tăng hoặc giảm kích thước chữ.
  • Màu sắc: chọn bằng bảng màu.
  • Độ đậm/nhạt: chỉnh bằng thanh kéo hoặc ô số. Giá trị càng cao watermark càng rõ.
  • Góc xoay: thường dùng -35 để watermark chéo.
  • Vị trí: mặc định giữa trang, có thể chọn các góc.
Khuyến nghị: watermark nên đủ nhìn thấy nhưng không che nội dung chính. Thường dùng độ mờ khoảng 0.12 đến 0.25.

7.Watermark logo

Watermark logo cho phép bạn đóng logo công ty, biểu tượng dự án hoặc dấu nhận diện lên PDF.

  1. Bật Watermark logo.
  2. Upload logo PNG hoặc JPG.
  3. Chỉnh độ đậm/nhạt bằng thanh kéo.
  4. Chọn file và trang đại diện để preview.
  5. Bấm Xem preview.
  6. Kéo logo để di chuyển.
  7. Kéo nút tròn ở góc logo để phóng to hoặc thu nhỏ.
  8. Nhập trang áp dụng nếu chỉ muốn đóng logo lên một số trang.

Ví dụ ô trang áp dụng:

  • 1-3: chỉ áp dụng cho trang 1, 2, 3.
  • 1,5,8-10: áp dụng cho trang 1, 5 và 8 đến 10.
  • Để trống: áp dụng cho tất cả trang.
Lưu ý: trang áp dụng được hiểu theo file PDF kết quả sau khi xuất/gộp.

8.Chữ ký

Tab Chữ ký hỗ trợ ký tay trực tiếp.

  • Chọn màu mực.
  • Chọn độ dày nét ký.
  • Dùng chuột hoặc cảm ứng để ký.
  • Bấm Lưu chữ ký để dùng khi đóng vào PDF.
  • Có thể ký một trang hoặc tất cả trang.

Nếu ký sai, dùng Undo hoặc Xóa rồi ký lại. Sau khi xuất file, hãy mở file để kiểm tra vị trí chữ ký trước khi gửi đi.

9.Page Manager

Page Manager giúp chỉnh từng trang trong một PDF.

  • Xem thumbnail: render ảnh thu nhỏ của từng trang.
  • Chọn tất cả: chọn toàn bộ trang.
  • Xoay 90°: xoay các trang đã chọn.
  • Xóa chọn: đánh dấu xóa trang.
  • Khôi phục: phục hồi trang đã xóa.
  • Xuất trang chọn: tạo file mới từ các trang đã chọn.

10.OCR

OCR dùng để nhận diện chữ từ PDF scan hoặc ảnh trong PDF.

  • Chọn file PDF.
  • Nhập trang cần OCR, ví dụ 1-3,5.
  • Chọn ngôn ngữ: Việt + Anh, Tiếng Việt hoặc English.
  • Bấm Chạy OCR.
  • Có thể tải kết quả TXT hoặc tạo PDF searchable.
Độ chính xác OCR phụ thuộc vào độ nét, độ nghiêng, font chữ, chất lượng scan và ngôn ngữ trong tài liệu.

11.QR Code

QR Code dùng để gắn link, mã xác minh, mã hồ sơ hoặc thông tin tra cứu vào PDF.

  • Bật Đóng QR vào PDF.
  • Nhập link hoặc mã xác minh.
  • Chọn vị trí QR.
  • Xuất PDF.

12.Batch Processor

Batch Processor xử lý nhiều PDF nhưng xuất riêng từng file, không gộp lại. Tính năng này phù hợp khi bạn cần đóng watermark hoặc đánh số trang cho nhiều tài liệu cùng lúc.

  • Watermark từng file.
  • Đánh số trang từng file.
  • QR từng file.
  • Ký từng file.

13.Live Editor

Live Editor cho phép đặt text trực tiếp lên một trang PDF.

  1. Chọn file PDF.
  2. Nhập số trang.
  3. Bấm Mở trang.
  4. Nhập nội dung text.
  5. Bấm + Text.
  6. Kéo object text đến vị trí mong muốn.
  7. Nhấp đúp object để xóa.

14.Reset dự án

Nút Reset dùng để đưa app về trạng thái ban đầu. Khi reset, app sẽ xóa file đang chọn, chữ ký, logo, OCR output, object editor, log, link tải và cấu hình lưu tạm.

Cẩn thận: reset không thể hoàn tác. Hãy tải file kết quả trước khi reset.

15.Kiểm tra file xuất

Trước khi gửi hoặc lưu file chính thức, bạn nên kiểm tra:

  • Đúng số trang.
  • Đúng thứ tự file.
  • Watermark không che nội dung quan trọng.
  • Logo nằm đúng trang và đúng vị trí.
  • Chữ ký đúng trang.
  • QR có quét được không.
  • Text OCR có đủ dùng không.
