Mục lục

Trước khi dùng File lớn PDF đặc biệt Watermark Chữ ký OCR Kiểm tra đầu ra Dữ liệu nhạy cảm Mẹo sử dụng

1.Trước khi dùng

PDF Fusion Smart Pro giúp xử lý PDF nhanh, nhưng bạn vẫn nên chuẩn bị file cẩn thận để có kết quả tốt nhất. Hãy đảm bảo file không bị hỏng, không bị khóa, có tên rõ ràng và đúng thứ tự cần xử lý.

  • Kiểm tra file mở được bằng trình đọc PDF thông thường.
  • Đặt tên file theo thứ tự nếu cần gộp.
  • Xác định trước trang cần tách hoặc trang cần đóng watermark.
  • Chuẩn bị logo/chữ ký có nền rõ và dung lượng vừa phải.

2.Lưu ý với file lớn

File PDF lớn, nhiều ảnh scan hoặc nhiều trang có thể tiêu tốn nhiều RAM. Vì app xử lý trong trình duyệt, tab có thể chậm nếu file quá nặng.

Khuyến nghị:

  • Xử lý theo từng phần nếu PDF quá dài.
  • Tách PDF trước rồi OCR từng nhóm trang.
  • Giảm số lượng file xử lý cùng lúc.
  • Đóng các tab nặng khác nếu thiết bị yếu.
  • Lưu file kết quả ngay sau khi xuất thành công.

3.PDF đặc biệt

Một số PDF có thể gây lỗi hoặc xuất kết quả không như mong muốn:

  • PDF có mật khẩu hoặc mã hóa.
  • PDF được tạo bởi phần mềm scan cũ.
  • PDF có nhiều kích thước trang khác nhau.
  • PDF có trang xoay, crop box phức tạp hoặc layer đặc biệt.
  • PDF chứa font nhúng lỗi hoặc cấu trúc không chuẩn.

Nếu gặp lỗi, hãy thử mở file bằng phần mềm PDF khác và xuất lại thành bản PDF mới rồi xử lý lại.

4.Lưu ý khi dùng watermark

Watermark nên hỗ trợ nhận diện tài liệu chứ không nên che mất nội dung chính. Khi dùng watermark chữ hoặc logo, hãy chú ý độ mờ, kích thước và vị trí.

  • Watermark chữ nên có opacity vừa phải, thường từ 0.12 đến 0.25.
  • Watermark logo nên dùng file PNG/JPG nhẹ.
  • Nếu logo che nội dung, hãy giảm opacity hoặc thu nhỏ logo.
  • Nếu chỉ muốn đóng logo vài trang, hãy nhập đúng dải trang áp dụng.
  • Với PDF nhiều kích thước trang, nên kiểm tra nhiều trang sau khi xuất.

5.Lưu ý khi dùng chữ ký

Chữ ký trong app là chữ ký hình ảnh được đặt lên PDF. Tính năng này phù hợp với xác nhận nội bộ, bản nháp, biểu mẫu đơn giản hoặc quy trình không yêu cầu chữ ký số.

Không nhầm lẫn: chữ ký vẽ tay trong app không phải chữ ký số có chứng thư số.
  • Không lưu chữ ký trên máy công cộng.
  • Bấm Reset sau khi ký tài liệu quan trọng.
  • Kiểm tra kỹ vị trí chữ ký trong file xuất.
  • Không dùng chữ ký của người khác nếu không có quyền.

6.Lưu ý khi OCR

OCR không thể đảm bảo chính xác tuyệt đối. Kết quả phụ thuộc nhiều vào chất lượng scan.

OCR có thể sai khi:

  • Trang bị mờ hoặc nghiêng.
  • Chữ quá nhỏ.
  • Ảnh bị bóng, nhiễu hoặc thiếu sáng.
  • Font chữ lạ.
  • Tài liệu có nhiều cột, bảng phức tạp hoặc dấu mờ.

Sau khi OCR, nên đọc lại kết quả nếu dùng cho mục đích quan trọng.

7.Kiểm tra file đầu ra

Sau khi xuất PDF, bạn nên mở file kết quả bằng trình đọc PDF và kiểm tra:

  • File có mở được không.
  • Số trang có đúng không.
  • Thứ tự trang có đúng không.
  • Trang bị xoay đúng chưa.
  • Watermark/logo/chữ ký đúng vị trí chưa.
  • QR code có quét được không.
  • OCR searchable có tìm kiếm được không.

8.Dữ liệu nhạy cảm

Nếu tài liệu chứa thông tin nhạy cảm như hợp đồng, giấy tờ cá nhân, hồ sơ tài chính, hồ sơ nhân sự hoặc tài liệu nội bộ, hãy dùng thiết bị đáng tin cậy và xóa trạng thái sau khi xử lý.

  • Không xử lý tài liệu nhạy cảm trên máy lạ.
  • Không để người khác truy cập trình duyệt khi đang mở app.
  • Không chia sẻ file kết quả khi chưa kiểm tra.
  • Không gửi file nhạy cảm vào kênh hỗ trợ công khai.

9.Mẹo sử dụng hiệu quả

  • Dùng tên file có số thứ tự: 01-main.pdf, 02-appendix.pdf.
  • Preview watermark logo trước khi xuất.
  • Với OCR, chạy thử 1 trang trước khi chạy nhiều trang.
  • Với batch, test 1 file trước rồi mới chạy nhiều file.
  • Giữ bản gốc của tài liệu để có thể xử lý lại nếu cần.
