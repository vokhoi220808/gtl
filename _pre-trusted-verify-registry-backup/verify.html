Kiểm tra thủ công

Dán link QR, link hash hoặc nội dung JSON certificate vào đây.

So hash PDF thật

Chọn PDF cần kiểm tra. Trang sẽ tính SHA-256 trong trình duyệt và so với hash trong link/certificate.

Chưa có PDF để kiểm tra.

Tải file .verify-certificate.json do PDF Fusion tạo sau khi xuất PDF.

Chưa có certificate.

Tải ảnh chụp QR lên. Nếu trình duyệt hỗ trợ BarcodeDetector, trang sẽ tự đọc link QR.

Cách hiểu kết quả

  1. QR checksum hợp lệ: link QR chưa bị sửa nhầm các trường ID/thời gian/nguồn.
  2. PDF SHA-256 khớp: file PDF tải lên giống từng byte với file đã được ghi trong link/certificate.
  3. Nếu hash lệch: PDF đã khác bản được chứng nhận, hoặc bạn đang dùng sai certificate/link.
