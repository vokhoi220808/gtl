✂️ Tách PDF theo dải trang MỚI

Mỗi dải cách nhau bằng dấu phẩy , — mỗi dải sẽ tạo 1 file PDF riêng
Ví dụ:
1-2,4-5 → xuất 2 file: trang 1→2 và trang 4→5 5-10 → 1 file trang 5 đến 10 1,3,5-8 → 3 file: trang 1, trang 3, trang 5→8
💡 Mỗi dải trang sẽ được xuất thành 1 file PDF độc lập, đặt tên theo dải trang.
